ABOUT US

    Abstract

  • This project is aimed at developing a Repository and search Engine for alumni of the college
  • which is of importance to a college.
  • Anyone can access the search Engine to know about any Alumni of that college.
  • This web application is used to manage the college alumni information.
  • The objective of this application is to allow old and new students of a university or college to communicate with each other.
  • This allows students to know about each other and their current activities.

    • Description

  • The Existing system is a computerized system but which is maintained at individual databases i.e. in excels sheets, it is a time delay process and maintaining all the records in Excel sheets is difficult. It doesnot provide multiple user accessibility and also doesnot have different user privileges.
  • The Proposed system is a computerized system but which is maintained at Centralized databases i.e. in automated forms it is a very fast process. And maintaining all the records in online systems database which makes it very easy to access and retrieve data from the database. If they want any record they can easily search all the records. It provides multiple user accessibility and also has different user privileges.
